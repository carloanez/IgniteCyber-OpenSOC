# Contributing to IgniteCyber OpenSOC

Thanks for your interest! This project is an open education/research lab.

## How to contribute
1. Fork the repo, create a feature branch: `git checkout -b feat/my-change`
2. Make changes with clear commits and docs
3. Run basic checks: `npm run lint:md` (optional) and `docker compose config`
4. Submit a Pull Request (PR) and link related issue(s)

## Contributor Agreement
By contributing, you agree to the **OpenSOC Contributor Agreement** (`OpenSOC-Contributor-Agreement.md`).

## ATT&CK Mapping
All detections, hunts, and labs must include MITRE ATT&CK mappings using the template in `docs/mitre-mapping/template.md`.

## Code of Conduct
All participants must follow the [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).
