Quick Start

Open a terminal and run the following commands:

cd IgniteCyber-OpenSOC

# Start the lab stack
sudo ./start_script.sh

# Check service health
./scripts/health-check.sh

# Open JupyterLab
./scripts/open-jupyter.sh

To stop the lab stack:

cd IgniteCyber-OpenSOC
./stop_script.sh
